# Welcome to OOWeb1 Documentation

OOWeb1 is a premium golf coaching platform built with Convex, designed to help golfers improve their game through video lessons, personal coaching, and performance tracking.

# Features

Core features of OOWeb1:

- 🎥 **Video Lesson Library** - Professional golf coaching videos
- 💬 **Coach-Student Chat** - Direct messaging with certified instructors
- 📹 **Video Uploads** - Submit swing videos for personalized feedback
- 📊 **User Dashboard** - Track your golfing progress
- 🎯 **Handicap Tracking** - Monitor and improve your handicap
- 💳 **Subscription Plans** - Free and Pro tiers with Stripe payments

## Tech Stack

- 🧩 **Convex**: Reactive, typesafe backend with authentication
- ⚡ **Vite**: Next-Gen Frontend Tooling
- 🎨 **TailwindCSS**: Utility-First CSS Framework
- 📐 **ShadCN**: Composable React components
- 🌙 **Easy Theming**: Dark/Light mode support
- 🗺️ **TanStack Router**: File-based routing
- 💳 **Stripe**: Subscription payments
- 📧 **Resend**: Email notifications

> [!NOTE]
> This is a template project for OOWeb1 golf coaching platform. Core features are planned but not yet implemented.

# Getting Started

```sh
# install dependencies
npm install

# set up Convex
npx convex dev --configure=new --once

# start development
npm start
```

You should be able to access your app at [http://localhost:5173](http://localhost:5173) 🎉
