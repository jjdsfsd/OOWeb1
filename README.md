# Features

Core features planned for OOWeb1:

- 🎥 **Video Lesson Library** - Browse and watch professional golf coaching videos
- 💬 **Coach-Student Chat** - Direct messaging between coaches and students
- 📹 **Video Uploads** - Students can submit swing videos for coach review
- 📊 **User Dashboard** - Track your golfing progress and statistics
- 🎯 **Handicap Tracking** - Monitor and improve your handicap over time
- 💳 **Subscription Plans** - Access premium content with flexible pricing

## Tech Stack

- ⚡ **Convex** - Reactive, typesafe backend
- ⚡ **Vite** - Next-Gen Frontend Tooling
- 🎨 **TailwindCSS** - Utility-First CSS Framework
- 📐 **ShadCN** - Composable React components
- 🗺️ **TanStack Router** - Simple Route Definitions
- 💳 **Stripe** - Subscription payments
- 📧 **Resend** - Email notifications

## Getting Started

```bash
npm install
```

## Architecture

This is a template project for OOWeb1 golf coaching platform. The codebase is structured with:

- `convex/` - Backend functions, schema, and API routes
- `src/` - Frontend React application
  - `src/routes/` - Page routes
  - `src/ui/` - UI components
  - `src/utils/` - Utility functions
